# Impedance Lab

Interactive single-page learner for **impedance** and the **reflection coefficient Γ**.

Open `index.html` in a browser — no build step, no backend.

## What it teaches

- Normalized impedance \(z = Z_L / Z_0\)
- Reflection coefficient \(\Gamma = (z - 1) / (z + 1)\)
- Polar form \(|\Gamma|\), \(\angle\Gamma\)
- VSWR, return loss, mismatch loss
- Reading the Smith chart (Γ plane)

## Features

- Drag the operating point on an interactive Smith chart
- Sliders for \(R\), \(X\), \(Z_0\) (Z → Γ mode)
- Sliders for \(|\Gamma|\), \(\angle\Gamma\) (Γ → Z mode)
- Live step-by-step calculation walkthrough
- Presets: matched, open, short, common loads
